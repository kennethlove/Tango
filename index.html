Tango

Faster, simpler Django content management

Download this project as a .zip file Download this project as a tar.gz file

Why Tango?

Because building content-driven web sites with Django is harder than it has to be. Tango is the result of more than six years building content-driven Django sites, including large publications. Like Django itself, it is designed to get a lot done, quickly and simply. Deadlines don't allow for fighting with the content manager.

Batteries included

Tango includes everything you need for a content site, including news & articles, blogs, photo galleries, video support, contact forms and more.

Is it ready for prime-time?

Honestly, no. Not yet. We're just getting started. Much more is coming soon.
